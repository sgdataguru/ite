The 5-Day Journey

Progressive learning from foundations to production deployment

1

Day 1: Foundations & Environment

Understanding WHY before HOW

Morning: Strategic Theory

  • Why data pipelines matter in Singapore
  • ETL vs ELT: The modern shift
  • Medallion Architecture (Bronze ‚Üí Silver ‚Üí Gold)
  • Modern data stack landscape

Afternoon: Hands-On Lab

  • Complete environment setup (Databricks, Docker, Airflow)
  • First data extraction exercise
  • Capstone project kickoff & team formation
Deliverable: Working dev environment + extracted sample dataset
2

Day 2: Data Extraction & Ingestion

Getting data from anywhere, reliably

Morning: Strategic Theory

  • Data source taxonomy (Files, APIs, Databases, Streams)
  • API extraction mastery with pagination & rate limiting
  • Batch vs Streaming decision framework
  • Data contracts & source documentation

Afternoon: Hands-On Lab

  • Multi-source extraction workshop (3 stations)
  • Build Bronze layer for capstone project
  • Error handling & recovery patterns
Deliverable: Bronze layer with data from 3 different sources
3

Day 3: Transformation & Data Modeling

From raw to ready: Silver & Gold layers

Morning: Strategic Theory

  • The dbt revolution: transforms as code
  • Data quality testing (unique, not_null, custom tests)
  • Dimensional modeling essentials (Star schema)
  • Bronze ‚Üí Silver ‚Üí Gold transformation patterns

Afternoon: Hands-On Lab

  • Build Silver layer with dbt models
  • Add comprehensive testing & documentation
  • Create Gold layer (fact & dimension tables)
Deliverable: Complete Bronze ‚Üí Silver ‚Üí Gold transformations with tests
4

Day 4: Orchestration & Monitoring

Making pipelines run themselves (reliably)

Morning: Strategic Theory

  • Why orchestration matters: from cron to Airflow
  • Airflow deep dive (DAGs, operators, dependencies)
  • Scheduling with cron expressions
  • Failure handling & alerting strategies

Afternoon: Hands-On Lab

  • Orchestrate capstone pipeline with Airflow DAG
  • Add comprehensive alerting (Slack/email)
  • Build observability dashboard with Streamlit
Deliverable: Scheduled DAG with monitoring dashboard & alerts
5

Day 5: Production & Teaching

From workshop to classroom

Morning: Production Readiness

  • Production checklist (security, reliability, observability)
  • CI/CD for data pipelines with GitHub Actions
  • Documentation & runbooks
  • Teaching adaptation workshop

Afternoon: Showcase & Synthesis

  • Capstone project presentations (8 min each)
  • Curriculum development workshop
  • Final synthesis & resource distribution
  • Certification ceremony
Deliverable: Production-ready pipeline + teaching materials + certificate
